Data Analysis
Full Syllabus

A structured roadmap from fundamentals to advanced analytics — with a natural pivot point into AI & Machine Learning.

Data Analysis path
AI/ML path
Foundation for both paths
PHASE 01
Weeks 1–4
Foundations of Data

Build the bedrock — data types, spreadsheet mastery, and statistical thinking. No prior experience needed.

DA
Introduction to Data Analysis
What is data? Types of data (structured, unstructured), data lifecycle, real-world use cases.
DA
Excel / Google Sheets
Formulas, pivot tables, VLOOKUP/XLOOKUP, data validation, charts & dashboards.
DA
Descriptive Statistics
Mean, median, mode, variance, standard deviation, distributions, skewness, kurtosis.
DA
Data Collection & Quality
Sources of data, scraping basics, missing values, outlier detection, data cleaning principles.
PHASE 02
Weeks 5–9
SQL & Databases

Query real databases. SQL is the most-used skill in data — master it deeply before moving on.

DA
SQL Basics
SELECT, WHERE, GROUP BY, ORDER BY, HAVING, aliases, DISTINCT, LIMIT.
DA
Joins & Subqueries
INNER, LEFT, RIGHT, FULL joins, CTEs, nested queries, correlated subqueries.
DA
Window Functions
ROW_NUMBER, RANK, DENSE_RANK, LAG/LEAD, PARTITION BY, running totals.
DA
Database Design Basics
Relational vs NoSQL, ERDs, normalization (1NF–3NF), indexing fundamentals.
PHASE 03
Weeks 10–15
Python for Data Analysis

Python unlocks the full data ecosystem. Learn to manipulate, explore, and visualize data programmatically.

BOTH
Python Fundamentals
Variables, loops, functions, list comprehensions, OOP basics, file I/O, error handling.
BOTH
NumPy
Arrays, vectorization, broadcasting, linear algebra ops — foundation for ML too.
BOTH
Pandas
DataFrames, merging, groupby, time series, apply/map, handling nulls, reshaping data.
DA
Data Visualization
Matplotlib, Seaborn, Plotly — histograms, scatter plots, heatmaps, interactive charts.
DA
Exploratory Data Analysis (EDA)
Profiling datasets, correlation analysis, distribution analysis, storytelling with data.
⚡ Pivot Point — End of Month 4
This is Where Your Path Branches

After completing Phase 3, you have a strong Python + data foundation. This is the ideal moment to decide: continue deep into Data Analysis (BI, dashboards, business intelligence, statistics) — or pivot into AI/Machine Learning (model building, deep learning, MLOps). Many topics in Phase 4 serve both paths.

→ Stay in Data Analysis
BI tools, advanced statistics, dashboards, storytelling
→ Divert to AI/ML
Scikit-learn, deep learning, NLP, MLOps
→ Hybrid Track
Data analyst who builds and deploys ML models
PHASE 04
Weeks 16–21
Statistics & Probability — Core for Both Paths

Regardless of which path you chose, this phase deepens your statistical reasoning — essential for analysis and model interpretation alike.

BOTH
Probability Theory
Conditional probability, Bayes' theorem, probability distributions (normal, binomial, Poisson).
BOTH
Inferential Statistics
Hypothesis testing, p-values, t-tests, chi-square, ANOVA, confidence intervals.
BOTH
Regression Analysis
Linear regression, multiple regression, logistic regression, assumptions and diagnostics.
BOTH
A/B Testing & Experimentation
Designing experiments, statistical power, sample size, interpreting results correctly.
PHASE 05A
Weeks 22–28 · DA Track
Business Intelligence & Advanced Analytics

For those on the Data Analyst track — master BI tools, dashboards, and reporting for business impact.

DA
Power BI / Tableau
Data modeling, DAX/LOD calculations, interactive dashboards, published reports.
DA
Advanced SQL & Analytics
Funnel analysis, cohort analysis, RFM segmentation, time-series SQL queries.
DA
Data Storytelling
Narrative-driven dashboards, choosing the right chart, presenting to non-technical audiences.
DA
KPIs & Metrics Frameworks
AARRR, North Star metric, OKRs, building analyst playbooks, stakeholder alignment.
PHASE 05B
Weeks 22–28 · AI/ML Track
Machine Learning Foundations

For those who pivoted to AI/ML — learn the core algorithms, model building workflow, and evaluation methodology.

ML
Scikit-learn Ecosystem
Pipelines, preprocessing, train/test split, cross-validation, model persistence.
ML
Supervised Learning
Decision Trees, Random Forest, XGBoost, SVM, KNN — classification & regression.
ML
Unsupervised Learning
K-Means, DBSCAN, PCA, dimensionality reduction, anomaly detection.
ML
Model Evaluation & Tuning
Metrics (F1, ROC-AUC, RMSE), confusion matrix, GridSearchCV, learning curves.
ML
Feature Engineering
Encoding, scaling, feature selection, handling imbalanced datasets, SHAP values.
PHASE 06
Weeks 29–36
Specialization & Production

Advanced tracks, real-world deployment, and portfolio building. Choose your specialization.

DA
dbt & Modern Data Stack
Data modeling in dbt, Snowflake/BigQuery, ELT pipelines, data cataloging.
ML
Deep Learning with PyTorch/TF
Neural networks, CNNs, RNNs, transformers, training loops, GPU usage.
ML
NLP & LLMs
Text preprocessing, embeddings, fine-tuning, RAG, prompt engineering, HuggingFace.
ML
MLOps
MLflow, model registries, Docker, FastAPI for serving, monitoring drift in production.
BOTH
Cloud Platforms
AWS/GCP/Azure essentials, S3, BigQuery, SageMaker, managed notebooks, IAM basics.
BOTH
Capstone Projects
End-to-end project from data ingestion to insight or deployed model — portfolio ready.
